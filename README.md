# GM - Messaging App

A clean, simple messaging app built with Flutter and Firebase.

## Features
- Simple messaging interface
- Photo sharing
- Firebase Firestore backend
- Real-time updates
- Cross-platform (Android, iOS, Web)

## Tech Stack
- Flutter/Dart
- Firebase (Auth, Firestore, Analytics)
- AdMob for monetization
